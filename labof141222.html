AJAX Implementation

The various elements required are :

  1.  The XMLHttpRequest readystatechange event
                This event is fired when the ready state property of the XMLHttpRequest changes
    Steps
  1. Create a document to be dynamically loaded
  2. Creating a XMLHttpRequest object named xhr
  3. call the on readystatechange event 
    Syntax: xhr.onreadystatechange()=function() {
                            }
  4. inside the anonymous function called, if the readyState is 4 and status is 200, implement the dynamically loaded document
  5. the open method of the httprequestobject is called that takes 3 parameters
    • method(get or post)
    • url, ie. file location
    • asynch, a boolean that is set either true or false (true=asynchronous)
  6. The responseText method is used for all formats that are not based on XML. It returns an exact representation of the response as a string. Plain text, (X)HTML, and JSON are all formats that use responseText. Once the response has completed loading, we just call the AJAX object, retrieve its value with responseText, and add it to the page.
